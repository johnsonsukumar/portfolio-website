Marketing Performance Dashboard

Built an automated, real-time marketing dashboard using SQL and Looker Studio to track CAC, ROAS, and channel-level ROI — giving stakeholders instant visibility into campaign performance.

ToolsSQL, Looker Studio
TypeDashboard & Reporting
Result60% Faster Reporting
📈

🎯 The Problem

The marketing team was spending 6+ hours every week manually compiling performance reports across Google Ads, Meta, and email campaigns in spreadsheets. Reports were always out of date by the time stakeholders read them.

There was no single source of truth for key metrics like ROAS, CAC, or channel-level contribution to revenue.

🔧 Tools Used

SQL (BigQuery) Looker Studio Google Sheets Google Ads API GA4

🛠️ Approach

  • Connected Google Ads, Meta Ads, and GA4 data into BigQuery using data transfer services
  • Wrote SQL queries to clean, join, and aggregate campaign data across all channels
  • Built calculated fields for CAC, ROAS, CPL, and channel contribution percentage
  • Designed a Looker Studio dashboard with date-range controls, channel filters, and drill-down views
  • Set up automated email delivery of the dashboard to stakeholders every Monday morning

💡 Key Learnings

  • Automating data pipelines into BigQuery eliminated manual errors and saved hours weekly
  • Stakeholders engaged far more with visual dashboards than static spreadsheet exports
  • Consistent metric definitions (CAC, ROAS) across teams prevents misaligned reporting
  • Looker Studio's blended data sources are powerful but require clean SQL upstream
